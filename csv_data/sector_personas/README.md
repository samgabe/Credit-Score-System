# Sector Personas CSV Pack

Persona-based dataset for demonstrating how scoring responds across economic profiles.

## Personas
- Salaried employee
- SME owner
- Gig worker
- Student
- Farmer
- Market trader
- Healthcare worker
- Transport operator
- Freelancer
- Public sector employee

## Files
- `users.csv` (10 users)
- `repayments.csv` (14 rows)
- `mpesa_transactions.csv` (14 rows)
- `payments.csv` (12 rows)
- `fines.csv` (6 rows)

## Use Cases
- Compare sector risk trends in analytics
- Demo persona-specific repayment behavior
- Validate mixed statuses (`on_time`, `late`, `defaulted`, `pending`, `failed`, `disputed`)
